How It Works

1

MONITOR

  • IoT sensors track AQI and cleanliness.
  • Drones run periodic visual inspections.
  • Data flows 24/7 to the Nagardarpan cloud.
2

ANALYZE & RANK

  • Algorithms compute live Environmental Scores.
  • Each ward gets a transparent ranking.
  • See trends and alerts in real time.
3

GET REWARDED

  • Improved ward scores earn CivicPoints.
  • Redeem at local partners and transit.
  • Fund and vote on community projects.
